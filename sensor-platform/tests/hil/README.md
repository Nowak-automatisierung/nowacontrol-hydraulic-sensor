# HIL Test Plan Baseline

Baseline fuer die Hardware-in-the-Loop-Teststruktur des nowaControl Hydraulic Sensor Projekts.

## Zweck

Dieses Verzeichnis dient als Ausgangspunkt fuer spaetere HIL-Tests mit echter Zielhardware.

## Geplanter Fokus

- Bring-up- und Flash-Validierung auf echter MG24-Hardware
- Sensor-nahe End-to-End-Pruefungen
- Sleep/Wake- und Power-bezogene Testablaeufe
- Zigbee-Pairing- und Kommunikationspruefungen mit echter Hardware
- Bootloader- und OTA-nahe Hardwaretests

## Aktueller Status

Aktuell ist nur die strukturelle Baseline vorhanden. Konkrete HIL-Skripte und Testablaeufe werden in spaeteren Arbeitspaketen ergaenzt.

## Hinweise

HIL-Tests setzen reale Zielhardware, reproduzierbare Testschritte und dokumentierte Testumgebungen voraus.
